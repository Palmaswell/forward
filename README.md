# Forward

> Check for automated AA and AAA conformance color contrast according to the WCAG
>

## Develop

* :rocket: Node.js `>=8`
* :cat2: yarn `>= 1.12`

```sh
  git clone https://github.com/Palmaswell/forward.git
  cd forward
  yarn
  
  yarn dev

  # In a second terminal window
  yarn test --watchAll
```
